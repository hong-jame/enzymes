## 특징

- **단일 파일 · 77KB** — 의존성 없음, 빌드 없음, 오프라인 동작
- **도해가 전부 인라인 SVG** — 벡터라 확대해도 선명하고, 다크모드에서 색이 함께 반전됨
- **모바일 우선** — 375px 기준으로 설계, 핀치 줌 지원
- 라이트/다크 모드 자동 대응

## 다루는 내용

1. **자발적이라는 말의 뜻** — 자유에너지 G, ΔG = ΔH − TΔS, 발열/흡열, 평형은 왜 죽음인가
2. **ATP: 세포의 잔돈** — 에너지 연계, 구조와 가수분해(−7.3 kcal/mol), ATP 회로
3. **효소란 무엇인가** — 활성화 에너지 장벽, 전이 상태, **효소는 Eᴬ만 낮춘다(ΔG 불변)**
4. **효소는 어떻게 작동하는가** — 활성 부위, 유도적합, 촉매 순환, Eᴬ를 낮추는 4가지 기작, 포화
5. **효소를 흔드는 것들** — 온도·pH, 보조인자와 조효소, 경쟁적/비경쟁적 억제, 효소의 진화
6. **세포는 효소를 어떻게 통제하는가** — 다른 자리 입체성 조절, 협동성, 되먹임 억제, 지역화
